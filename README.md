# Content Fabric App (Public Demo)

Бэкенд‑код для платформы поиска и обработки контента. Оркестрирует сбор данных из внешних источников (акторы Apify), рассчитывает метрики, определяет вирусные сигналы и управляет биллингом/подписками. 

## Структура директорий
- `Actions/` — действия Fortify/Jetstream и пользовательские/командные потоки.
- `Contracts/` — интерфейсы инфраструктуры и фичефлагов.
- `Enums/` — доменные enum (статусы биллинга, типы источников, статусы run, Apify).
- `Filament/` — админ‑панель, ресурсы и виджеты.
- `Http/` — контроллеры и middleware.
- `Jobs/` — джобы очереди (тонкие обертки над use‑cases).
- `Listeners/` — подписчики на события.
- `Mail/` — mailable‑шаблоны.
- `Models/` — Eloquent‑модели домена.
- `Notifications/` — каналы и уведомления.
- `Policies/` — политики авторизации.
- `Providers/` — сервис‑провайдеры и bindings контейнера.
- `Repositories/` — хелперы запросов и доступ к данным.
- `Services/` — ядро: адаптеры, use‑cases, интеграции.

**Интеграции**
- `Services/Apify/*` — клиент Apify, акторы, парсеры.
- `Services/Payment/PaymentManager.php` — интеграция YooKassa.
- `Services/GPTClient.php` — обертка над OpenAI.
